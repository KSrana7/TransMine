# TransMine
An Attention-based Transformer Architecture to Decode Chemical Transformations from Process Spectroscopic Data

## Requirements

- Python 3.9
- matplotlib == 3.3.4
- numpy == 1.20.3
- pandas == 1.1.5
- scikit_learn == 0.24.2
- seaborn==0.11.1
- h5py==3.1.0
- torch == 2.1.0

The dataset used in the work can be downloaded in the repo (https://github.com/zhouhaoyi/ETDataset).
